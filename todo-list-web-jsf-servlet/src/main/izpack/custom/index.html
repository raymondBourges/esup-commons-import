Procédure de démarrage de Example

Votre installation faite, vous pouvez maintenant via les raccourcis Windows démarrer l'application Example !

Pour lancer l'application
  • Cliquer sur Apache Tomcat depuis le menu "Démarrer" > "Programmes" > "Example" : une fenêtre s'ouvre, il vous faut alors cliquer sur start dans cette nouvelle fenêtre.
  • Se connecter à l'application depuis votre navigateur préféré par ce lien : http://localhost:8080/Example-web-jsf-servlet/


Pour stopper l'application
  • Cliquer sur Apache Tomcat depuis le menu "Démarrer" > "Programmes" > "Example" : une fenêtre s'ouvre, il vous faut alors cliquer sur stop dans cette nouvelle fenêtre.


Pour désinstaller l'application
  • Cliquez sur "Désinstaller Example" dans le menu "Démarrer" > "Programmes" > "Example".
