ESUP-Portail Example Application

esup-example is a web application that illustrates how to use the esup-commons library.

esup-example is 100% Java software and uses Spring for bean management, MyFaces for rendering and Hibernate for database management. It was written by Pascal Aubry for the ESUP-Portail consortium in 2006.

Documentation

Please refer to esup-commons.

Download

If you want to simply download the application and test it, go to the download section of the project page, hosted by SourceSup.

SVN access

Anonymous access

If you want to get the latest version of esup-example, get it from the SVN repository:

Connection type:   http or https
User: anonymous

Port: default
